# Um projeto para publicar trabalhos fotográficos que foram feitos por você.

Esse projeto é uma melhoria do primeiro projeto de exemplo e não funcional, agora com diversas melhorias e com cadastramento de usuarios e disponivel online para qualquer um usar e disponibilizar seus trabalhos de forma gratuita.

## Requerimentos

* [NPM](https://www.npmjs.com)
* [firebase-tools](https://firebase.google.com/docs/cli/#install_the_firebase_cli)

## Visão Geral do Projeto

Este é um projeto em [Firebase](https://firebase.google.com/)

Tecnologias e Frameworks usados são

* [Authentication](https://firebase.google.com/docs/auth/web/start) para permitir que os usuários façam login no app.
* [Cloud Firestore](https://firebase.google.com/docs/firestore/quickstart) para adicionar dados e ver os dados adicionados.
* [Cloud Storage](https://firebase.google.com/docs/storage/web/start) para armazenamento das imagens com alta disponibilidade.
* [Bootstrap](https://getbootstrap.com.br) para permitir um projeto responsivo para dispositivos na web.
