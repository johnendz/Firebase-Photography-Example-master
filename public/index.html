Bem vindo ao Fire Photos

Este projeto foi desenvolvido por @johnendz, utilizando as principais tecnologias do firebase como exemplo, espero que tenha aprendido algo.

O que seria Fire Photos?

Fire Photos é um projeto de exemplo usando tecnologias disponivel pelo Firebase, como Authentication, Firestore, Storage, para coisas rotineiras em um site, tendo codigos para todos exemplos basico para funcionar!

O projeto é gratuito posso baixar é usar para mim mesmo? sim, ele é open source e poderá usar em qualquer lugar (desde que não altere o footer), o firebase é uma otima ferramentas para você começar hoje.
